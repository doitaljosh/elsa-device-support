Device support package to optimize Linux distros for the LG V20's hardware.

## What works:
- display/touch
- internal speaker
- HiFi DAC
- microphones
- USB type-C OTG (partial)
- WiFi
- bluetooth (partial)
- battery/charging

## Installation:
# Extract, then ```cd``` to it. Install ```rsync``` then run ```rsync -avd patch_root/ /```
